## Text Styles for Figma
[Get a Figma copy](https://www.figma.com/community/file/1186085071546203382)

## Install
Install New UI Typography from your terminal via npm. Current version is 0.1.0

```
npm i @new-ui/typography
```

## New UI Typography

### Desktop Headings
